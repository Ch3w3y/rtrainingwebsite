Assignment Operators

R has multiple assignment operators, but the most commonly used are <- and =.

Using <-

# Assign a value to a variable
x <- 10

# Preferred for assignment in R scripts
y <- "Hello, R!"

Using =

# Assign a value to a variable
x = 10

# Commonly used in function arguments
plot(x = 1:10, y = 1:10)

Best Practice

Use <- for assignments in scripts and = for function arguments.

Data Types and Structures

R has several data types and structures, each suited for different tasks.

Data Types

# Numeric
x <- 42.5

# Integer
y <- as.integer(42)

# Character
z <- "Hello, R!"

# Logical
flag <- TRUE

Data Structures

# Vector
vec <- c(1, 2, 3)

# Matrix
mat <- matrix(1:9, nrow = 3)

# Data Frame
df <- data.frame(
  id = 1:3,
  name = c("Alice", "Bob", "Charlie"),
  score = c(90, 85, 88)
)

# List
lst <- list(
  name = "Alice",
  age = 25,
  scores = c(90, 85, 88)
)

Logical Values and Comparisons

Logical values are used for comparisons and conditional statements.

Logical Operators

# Equality
x == y

# Inequality
x != y

# Greater/Less than
x > y
x < y

# Greater/Less than or equal to
x >= y
x <= y

Combining Logical Values

# AND
x & y

# OR
x | y

# NOT
!x
