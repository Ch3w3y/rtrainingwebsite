Loops

Loops allow you to repeat tasks multiple times.

For Loops

# Basic for loop
for (i in 1:5) {
  print(i)
}

# Loop over a vector
numbers <- c(10, 20, 30)
for (num in numbers) {
  print(num * 2)
}

While Loops

# Basic while loop
count <- 1
while (count <= 5) {
  print(count)
  count <- count + 1
}

Conditional Statements

Conditional statements allow you to execute code based on specific conditions.

If-Else Statements

# Basic if-else
x <- 10
if (x > 5) {
  print("x is greater than 5")
} else {
  print("x is less than or equal to 5")
}

Case When

# Using case_when from dplyr
library(dplyr)
data <- tibble(
  score = c(90, 75, 60, 45),
  grade = case_when(
    score >= 90 ~ "A",
    score >= 80 ~ "B",
    score >= 70 ~ "C",
    score >= 60 ~ "D",
    TRUE ~ "F"
  )
)

Functional Programming with purrr

The purrr package provides tools for functional programming in R.

Using map()

# Apply a function to each element of a list
library(purrr)
numbers <- list(1, 2, 3, 4, 5)
squared <- map(numbers, ~ .x^2)

Using map2()

# Apply a function to pairs of elements from two lists
list1 <- list(1, 2, 3)
list2 <- list(10, 20, 30)
sums <- map2(list1, list2, ~ .x + .y)
