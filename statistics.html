Descriptive Statistics

Descriptive statistics summarize and describe the main features of a dataset.

Basic Summary Statistics

# Summary of a dataset
summary(data)

# Mean, median, and standard deviation
mean(data$column_name, na.rm = TRUE)
median(data$column_name, na.rm = TRUE)
sd(data$column_name, na.rm = TRUE)

Quantiles

# Calculate quantiles
quantile(data$column_name, probs = c(0.25, 0.5, 0.75), na.rm = TRUE)

Frequency Tables

# Frequency of values
table(data$column_name)

# Proportions
prop.table(table(data$column_name))

Correlation Analysis

Correlation measures the strength and direction of the relationship between two variables.

Calculating Correlation

# Pearson correlation
cor(data$x, data$y, method = "pearson")

# Spearman correlation
cor(data$x, data$y, method = "spearman")

# Kendall correlation
cor(data$x, data$y, method = "kendall")

Correlation Matrix

# Correlation matrix for multiple variables
cor(data[, c("x", "y", "z")], use = "complete.obs")

Hypothesis Testing

Hypothesis testing is used to determine whether there is enough evidence to reject a null hypothesis.

t-Test

# One-sample t-test
t.test(data$column_name, mu = 0)

# Two-sample t-test
t.test(data$column_name ~ data$group)

# Paired t-test
t.test(data$before, data$after, paired = TRUE)

Chi-Square Test

# Chi-square test for independence
chisq.test(table(data$column1, data$column2))

ANOVA

# One-way ANOVA
aov_result <- aov(data$column_name ~ data$group)
summary(aov_result)
