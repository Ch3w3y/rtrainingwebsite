Handling Missing Values

Missing values are common in datasets. R provides several ways to handle them.

Identifying Missing Values

# Check for missing values
is.na(data)  # Returns TRUE for missing values

# Count missing values in each column
colSums(is.na(data))

# Check if any missing values exist
any(is.na(data))

Removing Missing Values

# Remove rows with missing values
data_cleaned <- na.omit(data)

# Remove rows with missing values in specific columns
data_cleaned <- data[!is.na(data$column_name), ]

Replacing Missing Values

# Replace missing values with a specific value
data$column_name[is.na(data$column_name)] <- 0

# Replace missing values with the column mean
data$column_name[is.na(data$column_name)] <- mean(data$column_name, na.rm = TRUE)

Handling Duplicate Rows

Duplicate rows can skew analysis results. R provides tools to identify and remove duplicates.

Identifying Duplicates

# Check for duplicate rows
duplicated(data)

# View duplicate rows
data[duplicated(data), ]

Removing Duplicates

# Remove duplicate rows
data_cleaned <- data[!duplicated(data), ]

Handling Data Types

Understanding and managing data types is crucial for effective data cleaning and manipulation.

Checking Data Types

# Check the type of a variable
class(data$column_name)

# Check the structure of the dataset
str(data)

Converting Data Types

# Convert to numeric
data$column_name <- as.numeric(data$column_name)

# Convert to character
data$column_name <- as.character(data$column_name)

# Convert to factor
data$column_name <- as.factor(data$column_name)

Working with Factors

# View levels of a factor
levels(data$column_name)

# Reorder levels
data$column_name <- factor(data$column_name, levels = c("Low", "Medium", "High"))

# Add new levels
data$column_name <- factor(data$column_name, levels = c(levels(data$column_name), "Very High"))
